# FAQ

## Apa itu Nexotao?

Reseller akses model AI. Dengan satu akun dan satu saldo Rupiah, kamu bisa memakai
model top untuk teks — lewat web chat, API, atau Claude Code, tanpa kartu kredit
luar negeri.

## Apakah aman?

Permintaanmu diteruskan ke penyedia model resmi untuk diproses. Kami tidak menjual
datamu. Detail ada di kebijakan privasi di [nexotao.com](https://nexotao.com).

## Apakah saldo bisa hangus?

Tidak. Saldo tidak kedaluwarsa kecuali dinyatakan lain, dan tetap bisa dipakai
sampai habis.

## Bagaimana cara membayar?

Top-up lewat QRIS. Tanpa langganan — kamu hanya membayar saldo yang diisi, dan
terpotong sesuai pemakaian.

## Bisa dipakai untuk Claude Code?

Bisa. Nexotao kompatibel dengan endpoint Anthropic, jadi Claude Code bisa diarahkan
ke Nexotao dan dibayar pakai Rupiah. Lihat [Claude Code](/claude-code).

## Apakah ada refund?

Top-up bersifat final dan saldo umumnya non-refundable. Untuk kasus seperti
dobel-charge atau kegagalan sistem, lihat kebijakan refund di
[nexotao.com](https://nexotao.com).

## Berapa harganya?

Tarif live ada di [Model & Harga](/model-harga). Saat ini jauh di bawah harga
official — anggap harga promo.

## Kenapa DeepSeek kadang balas 429 / "server at capacity"?

Karena request-nya bukan untuk DeepSeek. `DeepSeek-V4-Flash` diposisikan untuk
kerjaan **volume tinggi & hemat** — klasifikasi, ekstraksi, chat pendek — dan
kapasitasnya kecil, sekitar **20k token/menit (TPM)** per backend. Satu request
**agentic/coding atau konteks besar** gampang menembus ~50k token input, sudah
melampaui jatah satu backend, jadi balas `429 server at capacity, retry shortly`
— sebanyak apa pun backend paralel tidak menolong, karena satu request tidak
bisa dipecah antar-backend.

Solusinya bukan retry berulang, tapi **pakai model yang tepat**: untuk
coding/agentic dan konteks besar, pilih langsung id model **Claude
(Opus/Sonnet)**, misalnya `claude-opus-4-8`. DeepSeek tetap pilihan terbaik
untuk beban volume/murah.
