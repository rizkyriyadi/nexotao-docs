# Playground

Coba Nexotao langsung di sini — tanpa menulis kode. Tempel **API key** kamu, pilih
**model**, tulis **pesan**, lalu klik **Kirim**. Kamu akan lihat jawaban model beserta
**biaya Rupiah**-nya.

Belum punya key? Buat gratis di dashboard **[nexotao.com](https://nexotao.com)** (butuh
saldo untuk memanggil model). Baru pertama kali? Baca [Konsep Dasar](/konsep) dulu.
