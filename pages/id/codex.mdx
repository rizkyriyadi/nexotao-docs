# Codex CLI

**Codex** adalah coding agent terminal dari OpenAI. Codex berbicara format
**OpenAI-compat**, jadi arahkan ke endpoint `https://api.nexotao.com/v1` dan pakai
model OpenAI-compat Nexotao: `gpt-5.6-sol`, `gpt-5.6-terra`, `gpt-5.6-luna`,
`gpt-5-mini`, `grok-4.3`, `DeepSeek-V4-Pro`, atau `DeepSeek-V4-Flash`.

> Untuk model **Claude**, pakai [Claude Code](/claude-code) (format Anthropic). Codex
> tidak punya mode Anthropic.

## 1. Install Codex

Ikuti dokumentasi resmi OpenAI Codex. Cek instalasi:

```bash
codex --version
```

## 2. Set API key sebagai environment variable

Codex membaca key dari env var (bukan disimpan di file config).

```bash
# macOS / Linux — tambahkan ke ~/.zshrc atau ~/.bashrc
export OPENAI_API_KEY="sk-nexo-..."
```

```powershell
# Windows PowerShell
setx OPENAI_API_KEY "sk-nexo-..."
```

## 3. Daftarkan Nexotao sebagai provider

Buka (atau buat) `~/.codex/config.toml` lalu isi:

```toml
# ~/.codex/config.toml
model = "gpt-5.6-sol"         # model default; ganti sesuai kebutuhan
model_provider = "nexotao"
forced_login_method = "api"

[model_providers.nexotao]
name = "Nexotao"
base_url = "https://api.nexotao.com/v1"   # WAJIB pakai /v1
env_key = "OPENAI_API_KEY"                # nama env var berisi sk-nexo-...
wire_api = "responses"                    # WAJIB "responses"; "chat" sudah dihapus Codex
```

## 4. Jalankan

Dari direktori proyekmu:

```bash
codex
```

Ganti model kapan saja lewat flag:

```bash
codex -m grok-4.3
codex -m DeepSeek-V4-Pro
```

## Catatan penting

- **`base_url` harus berakhiran `/v1`** — Codex menambahkan `/responses` sendiri.
- **`wire_api = "responses"`** wajib. Codex versi terbaru sudah **menghapus**
  `wire_api = "chat"`, dan Nexotao mengekspos endpoint Responses (`/v1/responses`)
  dengan streaming penuh — jadi Codex jalan normal.
- **Provider ID** (`nexotao`) tidak boleh memakai nama cadangan seperti `openai`,
  `ollama`, atau `lmstudio`.
- Key **tidak** disimpan di `config.toml`; hanya nama env var-nya (`env_key`).

Biaya dipotong dari saldo Rupiah per token. Lihat [Penagihan & Harga](/penagihan).
