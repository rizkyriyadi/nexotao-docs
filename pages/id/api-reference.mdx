# API Reference

Base URL: `https://api.nexotao.com`. Autentikasi lewat API key — lihat
[Autentikasi](/autentikasi).

## Endpoint

| Method | Path | Fungsi |
| --- | --- | --- |
| `GET` | `/models` | Katalog harga publik untuk situs (tanpa key) |
| `GET` | `/v1/models` | Daftar model format OpenAI (butuh key) — dipakai n8n dsb. |
| `GET` | `/v1/models/{id}` | Detail satu model format OpenAI (butuh key) |
| `POST` | `/v1/chat/completions` | Chat format OpenAI (DeepSeek dan Claude) |
| `POST` | `/v1/responses` | Format OpenAI Responses — mendukung streaming; dipakai Codex CLI |
| `POST` | `/v1/messages` | Chat format Anthropic (model Claude) |

## Header respons

Setiap permintaan berbayar mengembalikan:

| Header | Arti |
| --- | --- |
| `X-Cost-Rp` | Biaya aktual permintaan dalam Rupiah |
| `X-Request-Id` | Id permintaan untuk pelacakan |

## Kode error

| Status | Arti | Tindakan |
| --- | --- | --- |
| `400` | Permintaan tidak valid | Periksa body/parameter |
| `401` | API key salah atau dicabut | Buat / pakai key yang aktif |
| `402` | Saldo kurang | Top-up saldo Rupiah |
| `429` | Terlalu banyak permintaan | Coba lagi dengan backoff |

Body error berformat JSON, mis.:

```json
{ "error": "insufficient balance" }
```

## GET /models

Mengembalikan daftar model aktif beserta harga jual (tanpa data biaya internal).
Dipakai oleh tabel harga live di halaman [Penagihan](/penagihan) dan
[Model & Harga](/model-harga).

```bash
curl https://api.nexotao.com/models
```

Setiap item memuat: `model`, `display_name`, `tier`, `provider`, `supports_vision`,
`modality`, dan kolom harga `*_micro` (micro-rupiah; 1 Rp = 1.000.000 micro).
