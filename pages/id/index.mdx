# Pengantar Nexotao

Nexotao adalah reseller akses model AI. Dengan **satu akun dan satu saldo Rupiah**,
kamu bisa memakai model top untuk teks — lewat web chat, API, atau Claude Code.
Tanpa kartu kredit luar negeri, tanpa langganan.

## Cara kerja

1. Isi saldo Rupiah lewat QRIS.
2. Buat API key di dashboard.
3. Panggil endpoint Nexotao dengan SDK yang sudah kamu kenal — biaya dipotong dari
   saldo sesuai pemakaian.

## Base URL

```
https://api.nexotao.com
```

## Kompatibel Anthropic & OpenAI

Nexotao mengekspos endpoint yang kompatibel dengan kedua format, jadi kamu cukup
mengganti base URL dan API key pada tool yang sudah ada:

- **OpenAI** — `POST /v1/chat/completions` — untuk **semua model, termasuk Claude**.
  Kalau ragu, pakai ini.
- **Anthropic** — `POST /v1/messages` — hanya untuk model Claude, format Anthropic native.

Panduan memilih ada di **[Endpoint & Format](/endpoint)**. Bingung menyambungkan tool
coding-mu? Semua panduan step-by-step ada di **[Integrasi](/integrasi)** — Claude Code,
Codex, VS Code (Cline, Roo Code, Continue, Kilo Code), Aider, OpenCode, dan pola
universal untuk tool lain.
