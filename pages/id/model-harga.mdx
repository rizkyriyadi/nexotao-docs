# Model & Harga

**Sama modelnya dengan Anthropic dan OpenAI langsung.** Bedanya: bayar per
token pakai QRIS. Tanpa kartu USD, tanpa langganan bulanan, tanpa FX card fee.

## Nama tampilan vs Model ID

Nama seperti **GPT-5.6 Sol** adalah **nama tampilan** untuk dibaca manusia.
Saat memanggil API, gunakan **Model ID** persis seperti `gpt-5.6-sol`.
Model ID bersifat **case-sensitive**: jangan mengubah huruf besar/kecil dan
jangan memakai nama tampilan sebagai nilai `model`.

Lihat semua Model ID langsung dari katalog live:

```bash
curl -s https://api.nexotao.com/models | jq -r '.models[].model'
```

Contoh pemetaan:

| Nama tampilan | Model ID untuk API |
| --- | --- |
| GPT-5.6 Luna | `gpt-5.6-luna` |
| GPT-5.6 Terra | `gpt-5.6-terra` |
| GPT-5.6 Sol | `gpt-5.6-sol` |

## Headline per 100 juta token

Angka bulat marketing untuk memperkirakan skala. Dihitung dari campuran 30%
token input / 70% token output — mendekati kebiasaan aplikasi produk.
Tarif sebenarnya per model tetap dihitung per token; lihat tabel live di bawah.

| Model              | Headline per 100M token | Positioning                                                    |
| ------------------ | ----------------------- | -------------------------------------------------------------- |
| GPT-5.6 Sol        | **Rp 700 ribu**         | Tier GPT-5.6 tertinggi; tarif input dan output sama rata. |
| GPT-5.6 Terra      | **Rp 765 ribu**         | Seimbang untuk coding, analisis, dan workflow agent. |
| GPT-5.6 Luna       | **Rp 399 ribu**         | Efisien untuk kerja harian dan volume tinggi. |
| Claude Opus 4.8    | **Rp 15 juta**          | Model reasoning terkuat Anthropic. **≈ 6–7× lebih murah** dari tarif list Anthropic. |
| gpt-5-mini         | **Rp 500 ribu**         | Termurah OpenAI — pas untuk classifier, extract, batch. |
| Claude Sonnet 4.6  | *segera*                | Cepat + pintar untuk chat produk volume tinggi. Menunggu aktivasi katalog. |
| Claude Haiku 4.5   | *segera*                | Tercepat + termurah dari Anthropic. Menunggu aktivasi katalog. |

Rasio "vs Anthropic list" sengaja ditampilkan sebagai kelipatan, bukan angka
absolut, supaya tetap akurat ketika Anthropic mengubah harga list mereka.

> **Pilih model sesuai kerjaannya.** Untuk beban **volume tinggi & hemat** —
> klasifikasi, ekstraksi, chat pendek — pakai model murah seperti
> `DeepSeek-V4-Flash` atau `gpt-5-mini`. Untuk **coding/agentic & konteks
> besar**, pakai **Claude (Opus/Sonnet)**: DeepSeek berkapasitas kecil
> (~20k token/menit), jadi satu request agentic/large-context bisa kena
> `429 server at capacity`.

## Tarif live per juta token

Selain headline di atas, tiap model punya tarif per juta token untuk **input**,
**output**, dan (untuk Claude/GPT) **cache-read + cache-write**. Nilai di sini
diambil langsung dari `GET /models` — selalu sinkron dengan katalog live,
tanpa harga yang di-hardcode di halaman.

## Cara membaca tarif

- **Teks** — tarif per 1 juta token, terpisah untuk input dan output.
- **Cache (Claude/GPT)** — cache-read jauh lebih murah dari input biasa; cache-write
  sedikit lebih mahal dari input biasa. Detail di [Penagihan & Harga](/penagihan).
- **Headline per 100M token** — perkiraan cepat, bukan tarif tagihan. Yang
  ditagih adalah `jumlah_token × tarif_model_saat_itu`.

## Kenapa "per 100 juta token"?

Model teks modern mengukur pemakaian dalam jutaan token per proyek. Angka
`per 100 juta token` = urutan skala yang tepat untuk membandingkan biaya
antar-model dengan cepat, tanpa terganggu oleh titik desimal pada tarif
per-juta-token.

Detail penagihan, cache, dan billing per request ada di [Penagihan & Harga](/penagihan).
