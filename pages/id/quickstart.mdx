# Quickstart

Dari nol sampai jalan dalam beberapa menit.

## 1. Daftar

Buat akun gratis di [nexotao.com](https://nexotao.com). Cukup email dan kata sandi.

## 2. Top-up saldo

Isi saldo Rupiah mulai Rp 10.000 lewat **QRIS** atau **crypto (USDT / USDC)**. Saldo
tidak kedaluwarsa dan dipotong per token sesuai pemakaian — tanpa langganan. Isi
Rp 150.000+ dapat bonus saldo **+6%**, Rp 500.000+ dapat **+15%**.

## 3. Buat API key

Di dashboard, buka **API Keys** lalu buat key baru. Bentuknya `sk-nexo-...`. Simpan
baik-baik; key hanya ditampilkan penuh sekali.

## 4. Request pertama

Format Anthropic (`/v1/messages`):

```bash
curl https://api.nexotao.com/v1/messages \
  -H "x-api-key: sk-nexo-..." \
  -H "anthropic-version: 2023-06-01" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-opus-4-8",
    "max_tokens": 256,
    "messages": [{"role": "user", "content": "Halo"}]
  }'
```

Format OpenAI (`/v1/chat/completions`):

```python
from openai import OpenAI

client = OpenAI(base_url="https://api.nexotao.com/v1", api_key="sk-nexo-...")

resp = client.chat.completions.create(
    model="gpt-5.6-terra",
    messages=[{"role": "user", "content": "Halo"}],
)
print(resp.choices[0].message.content)
```

```javascript
import OpenAI from "openai"

const client = new OpenAI({ baseURL: "https://api.nexotao.com/v1", apiKey: "sk-nexo-..." })

const resp = await client.chat.completions.create({
  model: "gpt-5.6-terra",
  messages: [{ role: "user", content: "Halo" }],
})
console.log(resp.choices[0].message.content)
```

Setiap respons menyertakan header `X-Cost-Rp` (biaya aktual dalam Rupiah) dan
`X-Request-Id`. Lihat [Penagihan & Harga](/penagihan) untuk detail.

## 5. Ganti model cukup ubah satu string

Panggilan yang sama, lebih murah atau lebih kuat — cukup ubah `model`:

- `DeepSeek-V4-Flash` — workhorse super murah untuk kerjaan **volume tinggi**:
  klasifikasi, ekstraksi, chat pendek. Bukan untuk agentic/coding atau konteks
  besar (kapasitas ~20k TPM — request berat bisa kena `429 server at capacity`).
- `claude-opus-4-8` — reasoning frontier untuk **coding/agentic & konteks besar**.
- `gpt-5.6-luna`, `gpt-5.6-terra`, `gpt-5.6-sol` — keluarga GPT-5.6 dari tier
  efisien, seimbang, sampai tier tertinggi dengan tarif Sol Rp 7.000 input/output.

Id model bersifat case-sensitive — tulis persis seperti tertera. Lihat semua model
dan harga hari ini di [nexotao.com/harga](https://www.nexotao.com/harga).
