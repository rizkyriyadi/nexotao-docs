# Integrasi

Nexotao bukan model baru yang harus kamu pelajari dari nol. Nexotao adalah **gerbang
(gateway)** ke model AI yang sudah kamu pakai — Claude, GPT, DeepSeek, Grok — hanya
saja **base URL** dan **API key**-nya diganti ke Nexotao, dan tagihannya dipotong dari
saldo Rupiah.

Artinya: tool coding apa pun yang bisa diarahkan ke endpoint kustom **bisa dipakai
dengan Nexotao**. Halaman ini menjelaskan pola integrasinya, lalu tautan step-by-step
per tool.

## Dua "mode" — cukup ingat ini

Semua integrasi turun ke salah satu dari dua mode. Kalau kamu paham ini, kamu bisa
menyambungkan tool apa pun, bahkan yang tidak ada di daftar bawah.

| Mode | Base URL | Header / env | Model | Dipakai untuk |
| ---- | -------- | ------------ | ----- | ------------- |
| **Anthropic-compat** | `https://api.nexotao.com` | `x-api-key` / `ANTHROPIC_AUTH_TOKEN` + `ANTHROPIC_BASE_URL` | `claude-opus-4-8`, `claude-sonnet-4-6`, … | Model **Claude** & tool yang berbicara format Anthropic (Claude Code, Cline mode Anthropic) |
| **OpenAI-compat** | `https://api.nexotao.com/v1` | `Authorization: Bearer` / `OPENAI_API_KEY` + `OPENAI_BASE_URL` | `gpt-5-mini`, `grok-4.3`, `DeepSeek-V4-Pro`, … | Model **OpenAI/DeepSeek/Grok** & tool format OpenAI (Codex, Aider, Continue, dll.) |

## Host Claude Code sendiri — Nexotao Orce

Mau Claude Code **plus** orkestrasi multi-agent dalam web console yang kamu host sendiri,
bukan cuma satu terminal? **[Nexotao Orce](/nexotao-orce)** adalah aplikasi open-source kami:
pasang, sambungkan ke Nexotao pakai key `sk-nexo-...`, lalu kendalikan agent dari perangkat
mana pun.

## Belum punya API key?

1. Daftar di [nexotao.com](https://nexotao.com) — cukup email.
2. Top-up saldo QRIS mulai Rp 10.000.
3. Buat API key `sk-nexo-...` di menu **API Keys**.

Detail ada di [Quickstart](/quickstart) dan [Autentikasi](/autentikasi).
