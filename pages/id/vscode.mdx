# VS Code

Extension AI populer di **VS Code** — Cline, Roo Code, Continue.dev, Kilo Code — semua
mendukung **provider kustom**. Cukup pilih tipe provider, isi base URL Nexotao, dan
tempel API key `sk-nexo-...`.

Ingat dua mode:

- **OpenAI Compatible** → base URL `https://api.nexotao.com/v1`, model `gpt-5-mini`,
  `grok-4.3`, `DeepSeek-V4-Pro`, `DeepSeek-V4-Flash`.
- **Anthropic** → base URL `https://api.nexotao.com`, model `claude-opus-4-8`,
  `claude-sonnet-4-6`, dll.

---

## Cline

<ToolLogo src="/logos/cline.svg" alt="Cline" size={26} />

1. Install extension **Cline** dari VS Code Marketplace.
2. Buka panel Cline → ikon **⚙️ Settings**.
3. Untuk model **OpenAI-compat**:
   - **API Provider** → `OpenAI Compatible`
   - **Base URL** → `https://api.nexotao.com/v1`
   - **API Key** → `sk-nexo-...`
   - **Model ID** → mis. `gpt-5-mini`
4. Untuk model **Claude**:
   - **API Provider** → `Anthropic`
   - Centang **Use custom base URL** → `https://api.nexotao.com` *(tanpa `/v1`)*
   - **API Key** → `sk-nexo-...`
   - **Model** → mis. `claude-opus-4-8`

> Jangan isi Base URL OpenAI Compatible dengan `https://api.openai.com/v1` — itu
> provider "OpenAI" yang berbeda.

## Roo Code

Roo Code adalah turunan Cline; langkahnya mirip.

1. Install extension **Roo Code**.
2. Panel Roo Code → ikon **gear** → **API Provider**.
3. **OpenAI Compatible**: Base URL `https://api.nexotao.com/v1`, API Key `sk-nexo-...`,
   Model mis. `gpt-5-mini`.
4. **Anthropic**: aktifkan **Use custom base URL** → `https://api.nexotao.com`, API Key
   `sk-nexo-...`, Model `claude-opus-4-8`.

> Roo Code memakai **native tool calling** untuk jalur OpenAI Compatible. Model yang
> dipilih harus mendukung function/tool calling — `gpt-5-mini` dan model Nexotao
> lainnya sudah mendukung.

## Continue.dev

<ToolLogo src="/logos/vscode.svg" alt="Continue in VS Code" size={26} />

Continue dikonfigurasi lewat file `~/.continue/config.yaml`.

```yaml
# ~/.continue/config.yaml
models:
  - name: GPT-5 Mini (Nexotao)
    provider: openai
    model: gpt-5-mini
    apiBase: https://api.nexotao.com/v1
    apiKey: sk-nexo-...

  - name: Claude Opus 4.8 (Nexotao)
    provider: anthropic
    model: claude-opus-4-8
    apiBase: https://api.nexotao.com
    apiKey: sk-nexo-...
```

- `provider: openai` → `apiBase` **pakai** `/v1`.
- `provider: anthropic` → `apiBase` **tanpa** `/v1`.
- File lama `config.json` masih terbaca tapi sudah *deprecated* — tulis `config.yaml`.

## Kilo Code

1. Install extension **Kilo Code**.
2. Panel Kilo → **Settings (gear)** → tab **Providers** → **Custom provider**.
3. Isi dialog:
   - **Provider ID** → `nexotao`
   - **Provider API** → `OpenAI Compatible` (untuk gpt/grok/deepseek) atau
     `Anthropic Messages` (untuk Claude)
   - **Base URL** → `https://api.nexotao.com/v1` (OpenAI-compat) atau
     `https://api.nexotao.com` (Anthropic)
   - **API key** → `sk-nexo-...`
   - **Models** → Kilo bisa auto-ambil dari `/v1/models`, atau masukkan ID manual.

> Untuk memakai Claude **dan** gpt/deepseek sekaligus, buat dua custom provider —
> satu `OpenAI Compatible`, satu `Anthropic Messages`.

## GitHub Copilot / lainnya

Tool yang tidak punya field provider kustom (mis. GitHub Copilot standar) belum bisa
diarahkan ke Nexotao. Untuk tool semacam itu, lihat [Tool lain (universal)](/kompatibel)
— pola umum untuk apa pun yang menerima base URL.

Biaya semua tool dipotong dari saldo Rupiah per token. Lihat [Penagihan & Harga](/penagihan).
