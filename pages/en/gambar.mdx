# Image Generation — retired

Image generation has been **retired** from Nexotao. The catalog is now text-only —
[Claude Opus 4.8 and DeepSeek V4 Pro](/model-harga) — and the
`/v1/images/generations` endpoint is no longer served.

See the [Changelog](/changelog) for the full catalog change, or go to
[Text & Chat](/teks) to make a request against the current catalog.
