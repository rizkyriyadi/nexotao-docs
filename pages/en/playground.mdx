# Playground

Try Nexotao right here — no code required. Paste your **API key**, pick a **model**,
type a **message**, then click **Send**. You'll see the model's reply along with its
**Rupiah cost**.

No key yet? Create one free in the **[nexotao.com](https://nexotao.com)** dashboard (you
need a balance to call a model). New here? Read [Core Concepts](/en/konsep) first.
